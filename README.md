## Introduction

Some functionality of markdown, like links have more than one way to be created and need some extra management to be sure they are working correctly, for example if you create a linklabel twice only the first one is used.

This extension aims to ease the management of links and other management topics regarding markdown.

## Features

The following features are currently supported.

### Commands

- MDMan: Create New Reference Link
    - Creates a new reference link and appends it to the end of the current markdown file. These links are not shown in preview and normally ignored by every parser.

- MDMan: Convert Inline Link to Reference Link
    - Lets you pick a inline link inside your document and changes it into a reference link, which is also appended at the end of the document.

### Settings

There are currently no settings configured.
